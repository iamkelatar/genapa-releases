# GENAPA Releases

This repository publishes the public customer installers for GENAPA Forge.

Use the latest release and choose exactly one native installer for your platform:

- Windows: `GENAPA.Installer.Windows.msi`
- Linux: `genapa-forge_<version>_amd64.deb` or `genapa-forge-<version>.x86_64.rpm`
- macOS: `genapa-forge-<version>-universal.pkg`

Start here:

- Latest public release: https://github.com/iamkelatar/genapa-releases/releases/latest

Public install contract:

- Download one native installer from the latest release for your OS.
- Do not use bundle archives, bootstrap scripts, portable builds, AppImage builds, `RELEASES*`, `assets.*.json`, or `releases.*.json` for customer installation.

Deprecation note:

- Older releases may still contain legacy quick-install and bundle instructions.
- Prefer the latest release page above for current customer installs.
